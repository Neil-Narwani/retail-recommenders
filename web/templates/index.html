About this Project

The purpose of this project is to visualize woodworking tool purchase data for companies using D3.

Our group has access to approximately 900,000 transactions from approximately 200,000 unique customers. This data is stored in two .csv files.

Anon_customers.csv contains customer information, including:

  • A unique customer ID for each customer
  • ZIP code
  • Birth month (many customers are missing this information)
  • Total Sales, or the total amount they have spent
  • Total Savings, or the total amount saved from discounts or coupons
  • Total Visits
  • The date of the customers most recent visit
  • ReferralCode (many customers are missing this information)

transactions.csv contains the transaction history of each customer, including:

  • ID, which is the ID of the item itself
  • CustomerID, which is the ID of the customer who purchased the item
  • Description, which is a description of the item
  • TransactionTime, which is the date and time of purchase
  • Price of the item including any discounts or sales
  • FullPrice of the item, which does not include sales or discounts
  • DepartmentName, or the name of the department this item comes from
  • DiscountCode, which is a non-zero value if a discount was applied
  • ReturnCode, which is a non-zero value if an item was returned

Business Applications

The business goal of the application is to focus on giving business owners an interactive tool to visualize trends and relationships within the data set. This would be divided into four primary focus areas:

  • Purchasing Trends
  • Geographical Data
  • Return & Discount Codes
  • Product Relationships & Predictive Modeling / Recommendation Engine
